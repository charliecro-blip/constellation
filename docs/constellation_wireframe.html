Brand direction

Constellation is an intimate, intelligent app for understanding the dynamics between people. The visual language should feel considered, literary, and slightly cosmic — not mystical or New Age. Think midnight observatory meets personal journal. Every design decision should serve the sense that the user is being given something real.

Color — base palette

Midnight backgrounds, warm cream text, amber gold for insight and action.

Color — relationship types

Type colors carry through badges, borders, constellation dots, and share cards.

Romantic — rose
Committed — deep rose
Family — teal
Friendship — blue
Professional — slate
Ex — grey

Typography

Display / interpretive prose: Cormorant Garamond (or similar humanist serif) — italic for pull quotes and The Bond; regular weight for body pattern text. This is where the app's intelligence lives.

UI chrome: Clean geometric sans (Inter, DM Sans) for navigation labels, timestamps, badges, type labels, and button text. The contrast between the two is intentional: serif = meaning, sans = navigation.

Voice & copy

Short, declarative, slightly literary. Never clinical, never mystical.

"Your world" — not "Your dashboard"
"The Bond" — not "Overview" or "Summary"
"What do you want to understand?" — not "Select topic"
"This keeps showing up" — not "Pattern detected"

Report prose should read like a thoughtful person wrote it after careful reflection, not like a horoscope and not like a data readout.

App icon & wordmark

Icon: A single six-point asterisk or three-node constellation mark on deep midnight. Simple enough to read at 44px. Avoid zodiac wheels and planet imagery — that's the category cliché.

Wordmark: "CONSTELLATION" in Cormorant Garamond, all-caps, generous tracking, set next to or beneath the asterisk mark. Gold on midnight for app launch and splash. Dark on cream for light-mode marketing contexts.

7 key screens — what each one does

Home / Hub
Saved relationships as a living star map. The user is the center node. Type badges color-code each connection. Recent insights below as return entry points. The constellation visual is literal — it names what the app is.

Type selection
First step of every new reading. Sets the doctrine layer for the entire report — which patterns surface, which suppress, and what vocabulary is used. Must happen before names are entered.

People picker / vault
Enter birth data once, it persists. Birth time clearly marked optional with explicit note on what's unavailable without it. The vault is the primary retention driver — people come back because their people are already here.

Report surface
"The Bond" opens every report — 2-3 sentences naming the core of the connection. Pattern cards follow with a five-dot signal strength. The surface hooks; depth is behind Read More. Type badge persists throughout.

Question mode
Users don't re-read reports. They return during a specific moment — a conflict, a distance, a question. Theme pills pull and reframe the relevant patterns without requiring a full re-read. The retention feature.

Share / Insight card
A single resonant pull-quote designed to share. User picks which pattern to surface, previews the card, and sends it — to the person in the report (converts a new user) or publicly. Every share is an acquisition opportunity.

Constellation view
Unlocks after 3+ readings. Shows what keeps appearing across all saved relationships. Not a personality test — a relational mirror. "Saturn keeps appearing across your closest connections." The self-pattern seeker's killer feature.
