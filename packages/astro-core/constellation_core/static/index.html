How it works
  1. Choose what kind of relationship this is.
  2. Enter two birth charts.
  3. Add relationship context if you want a more focused reading.
  4. Generate a Relationship Map.
  5. Save important relationships.
  6. Watch recurring patterns emerge in Constellation Patterns.

Constellation uses astrology as a symbolic pattern language. It does not decide whether a relationship is good, bad, doomed, or meant to be.
